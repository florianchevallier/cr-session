# Univers Générique — JDR

## Contexte
Ce transcript provient d'une partie de jeu de rôle (JDR). Le système de jeu spécifique n'est pas précisé. Analyse le contenu pour déduire les mécaniques utilisées.

## Éléments communs aux JDR
- **MJ (Maître du Jeu)** : narrateur et arbitre des règles
- **PJ (Personnages Joueurs)** : personnages contrôlés par les joueurs
- **PNJ (Personnages Non-Joueurs)** : personnages contrôlés par le MJ
- **Jets de dés** : résolution aléatoire des actions
- **Compétences / Caractéristiques** : attributs des personnages
- **Points de vie / Santé** : mesure de résistance aux dégâts

## Points d'attention pour l'analyse
- Identifie le système de jeu à partir des mécaniques observées
- Note les termes spécifiques au jeu utilisés par les joueurs
- Distingue les conversations hors-jeu (meta) des interactions en jeu
- Les jets de dés peuvent être exprimés de multiples façons
